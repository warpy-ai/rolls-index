# rolls-index
The rolls package index contains the necessary metadata for unroll to download rolls and resolve their dependencies.

The package index is available at:

https://index.script-lang.org/ as an HTTP API (sparse protocol)
https://github.com/warpy-ai/rolls-index as a git repository

Theres still documentation to be created about the index

The rolls index is maintained by [lgrodev](https://x.com/lgrodev)
